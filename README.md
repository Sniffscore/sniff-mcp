# sniff-mcp

**Agent-callable canine genomics.** An MCP server over the **Sniff Atlas** — breed-stratified allele frequencies for 9.67M variants across 188 dog breeds, calibrated AI pathogenicity (ESM2, AUC 0.935 vs OMIA), and an evidence-graded knowledge graph. Every answer carries a citation chain.

> **v0.0.1 is a name-reservation placeholder.** The full Streamable-HTTP MCP server is in active development. Dataset is live now: **https://doi.org/10.5281/zenodo.20566358** (CC-BY-4.0). Follow https://sniff.world.
